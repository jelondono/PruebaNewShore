Como el cliente futuramente va a estar consultando diferentes filtros, desarrolle un filtro que cumpla con todos los criterios futuros.

Los filtros existentes buscan por cualquier campo de la tabla y se pueden agregar nuevos campos sin ningun problema.

Asi no se deben crear varios campos de filtros para diferentes temas si no que en uno esta todo, se hace directamente con el dom asi que no sera pesado para la base de datos.
